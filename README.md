# TinderForFight
План сайта и стек технологий
1) В основу завяжем гугл карты, на них будут обозначения ближайших спортзалов и кружков по единоборствам.
2) Авторизированный пользователь сможет воспользоваться следующим функционалом:
   - мессенджер с соперником/тренером по спаррингу
   - поиск ближайшего спортзала с возможностью провести спарринг
   - возможность редактировать некоторые параметры для поиска лучшего соперника по спаррингу - масса, рост, возраст
3) Дизайн будет сделан в стиле фильма "Бойцовский клуб"
   
СТЭК ТЭХНОЛОГИЙ:
Html, CSS, PHP, JavaScript, MySql
